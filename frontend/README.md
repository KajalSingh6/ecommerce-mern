# Certify -> E-Commerce Frontend

## Project Overview
Certify is a modern e-commerce frontend application built using React and Vite.It allows users to browse products, view products details, add items to cart, place orders, and manage their accounts.

## Tech Stack 
-React
-Vite
-Tailwind CSS
-Axios
-React Router DOM 
-Context API
- React Icons
- React Toastify

## Features
-User authentication (Login/Signup)
-Product listing & product details
-Cart functionality
-Order placement
-Responsive UI using Tailwind CSS

## Folder Structure
src/
 ├── components
 ├── pages
 ├── context
 ├── assets
 ├── App.jsx
 └── main.jsx

 
## How to Run Project
1. Clone the repository
2. Install dependencies
npm install
3. Start development server 
npm run dev

## Environment Variables
Create a `.env` file and add:
VITE_BACKEND_URL = your_backend_url
