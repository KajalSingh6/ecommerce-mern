# Certify - Admin Panel (Frontend)

## Overview
This is the Admin Panel for the Certify E-commerce project built with React and Vite.

## Features
-Add, edit, and delete products
-View orders and their details
-Manage users and roles


## Tech Stack
-Ract.js
-Vite
-Tailwind CSS
-Axios for API calls
-React Router
- React Icons
- React Toastify

## Getting Started
### Installation
```bash
# Clone the repository
git clone <your-repo-url>

# Go to admin folder
cd admin

# Install dependencies
npm install

#Start devlopment server
npm run dev
```

# Environment Variables

Create a .env file in the root directory and add: 
VITE_BACKEND_URL=your_backend_url
