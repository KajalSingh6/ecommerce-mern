# MERN E-Commerce Application

## Project Overview
This is a full-stack E-commerce web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
Users can browse products, add items to cart, and place orders. Admin can manage products and orders.

## Tech Stack
Frontend: React.js, Tailwind CSS, Axios
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT

## Features

User Side
- User authentication (Login / Signup)
- Browse products
- Product details page
- Add to cart
- Place orders
- Order history

Admin Panel
- Add new products
- Update products
- Delete products
- View customer orders

## Folder Structure

frontend -> User website
backend -> API and database
admin ->  Admin dashboard
