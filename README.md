# MERN E-Commerce Application

## Project Overview

This is a **full-stack E-commerce web application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.
Users can browse products, view product details, add items to the cart, and place orders.
An **Admin Dashboard** is also available for managing products and customer orders.

---

## Tech Stack

Frontend

* React.js
* Tailwind CSS
* Axios
* React Router DOM

Backend

* Node.js
* Express.js

Database

* MongoDB

Authentication

* JSON Web Token (JWT)

---

## Features

### User Features

* User authentication (Signup / Login)
* Browse products
* Product details page
* Add to cart functionality
* Place orders
* Order history
* Responsive UI

### Admin Panel Features

* Add new products
* Update existing products
* Delete products
* Manage customer orders

## Installation

### 1. Clone the repository

```
git clone https://github.com/KajalSingh6/ecommerce-mern.git
```

### 2. Install dependencies

Frontend

```
cd frontend
npm install
npm run dev
```

Backend

```
cd backend
npm install
npm start
```

Admin Panel

```
cd admin
npm install
npm run dev
```
